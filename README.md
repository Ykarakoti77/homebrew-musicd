# Homebrew Tap for musicd

Install **musicd** using Homebrew.

## Install

```bash
brew tap ykarakoti77/musicd
brew install musicd
```

## Verify

```bash
dj --help
```

## About

musicd is a terminal-native music player daemon powered by mpv and yt-dlp.

Project repository: https://github.com/Ykarakoti77/musicd
